# Hyprland configuration files

## üì• Installation

```bash
git clone https://github.com/xorandd/Hyprland-Dots
cd Hyprland-Dots
./install.sh
```

After installation:
 - in nwg-look (open rofi using SUPER+D) select GTK theme `Flat-Remix-GTK-Grey-Darkest` and icon theme `Flat-Remix-Grey-Dark` because they
are not set by default, also set mouse cursor `Bibata-Modern-Ice` just to be sure.

General keybinds
| Keybind                               | Action                                                           |
|---------------------------------------|------------------------------------------------------------------|
| CTRL + ALT, Delete                    | Exit Hyrpland                                                    |
| CTRL + ALT, L                         | Lock screen (hyprlock)                                           |
| CTRL + ALT, P                         | Logout (wlogout)                                                 |
| SUPER + Return                        | Open terminal                                                    |
| SUPER + Q                             | Kill active window                                               |
| SUPER + E                             | Open file manager                                                |
| SUPER + B                             | Open browser                                                     |
| SUPER + SPACE                         | Toggle floating window                                           |
| SUPER + D                             | Open application menu                                            |
| SUPER + P                             | Pseudo (dwindle)                                                 |
| SUPER + J                             | Toggle split (dwindle)                                           |
| SUPER + ALT + R                       | Reload Waybar (script)                                           |
| SUPER + SHIFT + F                     | Fullscreen window                                                |
| SUPER + CTRL + F                      | Maximize window (fullscreen)                                     |
| SUPER + SHIFT + S                     | Screenshot region (script)                                       |
| SUPER + Left                          | Move focus left                                                  |
| SUPER + Right                         | Move focus right                                                 |
| SUPER + Up                            | Move focus up                                                    |
